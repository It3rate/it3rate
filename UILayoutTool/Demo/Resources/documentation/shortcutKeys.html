Modifier Key Description
  
 SpaceHand to drag around stage with mouse
 Ctrl(while dragging objects) duplicates objects
 ArrowsNudge selection (Shift for super nudge)
 Ctrl UpZ-Order up for selection
 Ctrl DownZ-Order down for selection
 DeleteDelete selection
 EscapeSelect none
  
ctrl optional on zoom commands
 1Zoom 100%
 2Zoom to Fit
 3Zoom to selection
 +Zoom in
 -Zoom out
  
 DToggle Debug mode
 NToggle Name mode (todo)
 RRepeat last command(s)
 ZUndo (ctrl optional)
 RRedo (ctrl optional)

CtrlSSave File

Ctrl/Flip Horizontal
Ctrl/Sh/Flip Vertical
Ctrl>Rotate Right 15 degrees
CtrlRotate Left 15 degrees
Ctrl/ShZRemove transforms on selection
Numpad align/distribute shortcuts are based on the visual location of the numbers
 Numpad 0Remove bonds from selection
 Numpad 1Align vertical centers (and add bonds if 'Smart Bonds' enabled)
 Numpad 2Align Bottom (and add bonds if 'Smart Bonds' enabled)
 Numpad 4Align Left (and add bonds if 'Smart Bonds' enabled)
 Numpad 5Distribute to last align (and add bonds if 'Smart Bonds' enabled)
 Numpad 6Align Right (and add bonds if 'Smart Bonds' enabled)
 Numpad 8Align Top (and add bonds if 'Smart Bonds' enabled)
 Numpad 9Align Horizontal Centers (and add bonds if 'Smart Bonds' enabled)

CtrlDownMove z-order Down one layer
Ctrl/ShUpMove to Top
Ctrl/ShDownMove to Bottom
